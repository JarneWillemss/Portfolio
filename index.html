Mijn projecten

Hier kan u enkele opdrachten en projecten bekijken die ik tijdens mijn bachelor traject heb gemaakt.

Stage Detection Engineer

Bij Cegeka liep ik drie maanden stage binnen het Detection Engineering-team. Tijdens deze periode werkte ik met SIEM-oplossingen zoals Splunk en Microsoft Sentinel, waarbij ik voornamelijk aan de slag ging met de querytalen SPL en KQL. Daarnaast heb ik mijn kennis verdiept in threat hunting en kreeg ik inzicht in de dagelijkse werking van een Cybersecurity Operations Center (SOC).

SOC Project

Voor mijn SOC-project, uitgevoerd in het kader van het vak Cybersecurity en Security Operations Center, heb ik zelf een volledig functionerend SOC opgebouwd. Dit SOC bestaat uit verschillende componenten, waaronder EDR met Suricata, een SIEM-oplossing met Wazuh, threat hunting en digital forensics met Velociraptor en penetratietesten met behulp van ART en Kali Linux.

Project 4.0

Voor het vak Project 4.0 werkte ik samen met studenten uit de richtingen AI en Application Development aan een project voor Kenter Belgium. We herbouwden en optimaliseerden hun bestaande webapplicatie.

Tijdens de ontwerpfase was ik Documentlead, waarbij ik verantwoordelijk was voor het structureren en opvolgen van alle projectdocumentatie. In de implementatiefase nam ik de rol van teamlead op mij, waarbij ik de samenwerking binnen het team coördineerde.

Daarnaast stond ik in voor het uittekenen en opzetten van de architectuur, evenals voor de beveiliging van de applicatie.

QR Quiz

In ons Systems Engineering-project hebben we een boeiende quiz-applicatie ontwikkeld waarmee studenten op een unieke en interactieve manier kunnen antwoorden. Door simpelweg een QR-code in de lucht te steken. Voor dit project ben ik vooral bezig geweest met grafieken en UI design.

Automatische Muizenval

Voor het vak IoT Essentials ontwikkelde ik een geavanceerde, slimme muizenval met behulp van Orange Pi.

De val reageert automatisch wanneer de ingebouwde sensor beweging detecteert binnen zijn bereik. Ze kan zichzelf resetten, en op het LCD-scherm wordt weergegeven hoe vaak de val is geactiveerd.

Dit was zonder twijfel een van mijn leukste projecten. Het was enorm boeiend om verschillende componenten via Arduino te laten samenwerken tot één goed functionerend geheel. Bovendien heb ik dankzij dit project veel bijgeleerd over elektriciteit en elektronica, een onderwerp dat me sindsdien nog meer fascineert.

Project Hosting

In mijn groepsproject voor het vak Project Hosting hebben we ons eigen hostingplatform gebouwd voor het draaien van applicaties, gebaseerd op onze eigen serverconfiguratie. Het concept is vergelijkbaar met het Okteto Cloud platform.

Dit was voor mij een compleet nieuw domein, wat het des te uitdagender en boeiender maakte.

Mijn focus lag voornamelijk op het automatiseren van het deployen van een applicatie zodra er een nieuwe virtuele machine (VM) wordt aangemaakt, wat het hele proces efficiënter maakt.

Docker Project

Voor het vak Linux Webservices ontwikkelde ik in een Docker-project een schaalbare webpagina met een achterliggende databank. Het doel was om een webtoepassing te ontwerpen en implementeren die dynamische inhoud kan weergeven.

Aangezien Docker een volledig nieuw concept voor mij was, vergde het project veel opzoekwerk en ‘trial and error’. Toch slaagde ik erin het project succesvol af te ronden.

Tijdens dit proces heb ik in korte tijd veel geleerd over het opzetten en beheren van containers met Docker en Docker Compose, een ervaring die mijn technische kennis aanzienlijk heeft versterkt.

Azure Project

In deze opdracht was het de bedoeling om event logs en prestatiegegevens van een Azure Windows VM door te sturen naar Azure Monitor, met als doel een efficiënte monitoring en analyse op te zetten.

Wat ik zo waardeer aan de opdrachten binnen het vak Cloud Platforms, is dat je sterk wordt aangemoedigd om zelfstandig oplossingen te zoeken. Voor deze opdracht had ik nauwelijks ervaring met Azure of cloudtechnologie in het algemeen.

Intussen heb ik een goed inzicht verworven in de basisprincipes van Azure, en specifiek in het effectief inzetten van Azure Monitor. Ik zie nu ook duidelijk het belang van goede monitoring binnen een cloudomgeving.

Stage

In Februari tot Mei 2025 heb ik 3 maanden stage gelopen bij Cegeka, hieronder kan u een overzicht vinden.

Introductie

Cegeka is een toonaangevend Europees IT-bedrijf dat organisaties helpt bij hun digitale transformatie. Met hoofdkantoor in Hasselt (België) biedt Cegeka oplossingen op het gebied van onder andere cloudservices, cybersecurity, softwareontwikkeling, dataplatformen en IT-outsourcing.

Tijdens mijn stage maakte ik deel uit van het Security Operations Team, meerbepaald het team van detection engineers. Hier heb ik actief heb meegewerkt aan het ontwikkelen en optimaliseren van detectielogica. Voor mijn stageopdracht heb ik detectieregels uit twee verschillende SIEM-omgevingen met elkaar afgestemd, met als doel het verkleinen van detection gaps. Zo heb ik bijgedragen aan het verbeteren van de interne processen binnen Cegeka en het versterken van de dienstverlening naar klanten toe.

Wil je hier meer over weten? Neem dan gerust een kijkje in mijn plan van aanpak en realisatiedocument!

Realisatie

Mijn stage bestond uit verschillende fasen. In de eerste fase maakte ik een inventarisatie van alle detectieregels en onderzocht ik welke regels overlap vertoonden tussen de twee SIEM-omgevingen. Deze analyse werd vastgelegd in een Excel-bestand genaamd 'Use-case library'. Omdat dit vertrouwelijke informatie bevat, heb ik een mock-up versie gemaakt die je kunt bekijken via de knop hieronder.

Tijdens de tweede en derde fase heb ik de regels onderling vergeleken en op elkaar afgestemd, zodat beide omgevingen dezelfde detectielogica hanteren. Dit proces werd uitgevoerd aan de hand van een specifieke procedure. Klik op de knop ‘Aligneer Procedure’ om deze te bekijken.

Tot slot stelde ik een reflectiedocument op over mijn stageperiode. Hierin beschrijf ik het verloop van de stage, de uitdagingen die ik ben tegengekomen en vooral, wat ik allemaal heb bijgeleerd bij Cegeka! Je kunt dit document bekijken via de knop ‘Reflectie'
