Muizenval

In mijn project voor IOT Essentials heb ik een geavanceerde muizenval gecreëerd.

Deze slimme muizenval sluit automatisch zodra de ingebouwde sensor beweging detecteert binnen zijn bereik. De val kan resetten, en het is mogelijk om op het LCD scherm informatie te vinden zoals hoe vaak de val al is afgegaan.

Dit was zonder twijfel één van mijn tofste projecten, het was heel tof om zelf met Arduino verschillende componenten samen te laten werken tot één functionerend geheel. Door dit project heb ik ook veel bijgeleerd over elektriciteit, iets wat ik super interessant vind.

Azure Project

In deze opdracht was het doel om event logs en prestatiegegevens van een Azure Windows VM te verzenden naar Azure Monitor voor effectieve monitoring en analyse.

Het toffe aan de opdrachten van het vak Cloud Platforms is dat je zelf moet uitvinden hoe je de opdracht tot een goed einde brengt, wat betekent dat je zelf nog veel moet uitzoeken. Voor ik deze opdracht startte wist ik weinig tot niets over de werking van Azure of cloud in het algemeen. Nu ken ik de basisprincipes van Azure, meerbepaald hoe je Azure monitor effectief kunt gebruiken en zie ik nu in waarom dit zo nuttig is.
