Mijn projecten

Hier kan u enkele opdrachten en projecten bekijken die ik tijdens mijn bachelor traject heb gemaakt.

Stage Detection Engineer

Bij Cegeka liep ik drie maanden stage binnen het detection engineering-team. Tijdens deze periode werkte ik met SIEM-oplossingen zoals Splunk en Microsoft Sentinel, waarbij ik voornamelijk aan de slag ging met de querytalen SPL en KQL.

Daarnaast heb ik mijn kennis verdiept in threat hunting en kreeg ik inzicht in de dagelijkse werking van een Cybersecurity Operations Center (SOC).

SOC Project

Voor mijn SOC-project, uitgevoerd in het kader van het vak Cybersecurity en Security Operations Center, heb ik zelf een volledig functionerend SOC opgebouwd.

Dit SOC bestaat uit verschillende componenten, waaronder EDR met Suricata, een SIEM-oplossing met Wazuh, threat hunting en digital forensics met Velociraptor en penetratietesten met behulp van ART en Kali Linux.

Project 4.0

Voor het vak Project 4.0 werkte ik samen met studenten uit de richtingen AI en Application Development aan een project voor Kenter Belgium. We herbouwden en optimaliseerden hun bestaande webapplicatie.

Tijdens de ontwerpfase was ik Documentlead, waarbij ik verantwoordelijk was voor het structureren en opvolgen van alle projectdocumentatie. In de implementatiefase nam ik de rol van teamlead op mij, waarbij ik de samenwerking binnen het team coördineerde.

Daarnaast stond ik in voor automatisatie via CI/CD pipelines, het uittekenen en opzetten van de architectuur en voor de beveiliging van de applicatie.

Security audit

Voor de gemeente Bocholt voerden we als team een security audit uit op hun digitale omgevingen, met als doel de beveiliging van hun systemen te optimaliseren.

Tijdens dit project heb ik mijn kennis van offensieve cybersecurity verder verdiept en waardevolle ervaring opgedaan met het proces dat penetratietesters doorgaans volgen.

Voor dit project stond ik vooral in voor het testen van de fysieke beveiliging en heb ik verschillende kwetsbaarheden gevonden in randapparatuur zoals printers.

Automatische Muizenval

Voor het vak IoT Essentials ontwikkelde ik een geavanceerde, slimme muizenval met behulp van Orange Pi.

De val reageert automatisch wanneer de ingebouwde sensor beweging detecteert binnen zijn bereik. Ze kan zichzelf resetten, en op het LCD-scherm wordt weergegeven hoe vaak de val is geactiveerd.

Dit was zonder twijfel een van mijn leukste projecten. Het was enorm boeiend om verschillende componenten te laten samenwerken tot één goed functionerend geheel.

Bovendien heb ik dankzij dit project veel bijgeleerd over elektriciteit en elektronica, een onderwerp dat me sindsdien nog meer fascineert.

Project Hosting

In mijn groepsproject voor het vak Project Hosting hebben we ons eigen hostingplatform gebouwd voor het draaien van applicaties, gebaseerd op onze eigen serverconfiguratie. Het concept is vergelijkbaar met het Okteto Cloud platform.

Dit was voor mij een compleet nieuw domein, wat het des te uitdagender en boeiender maakte.

Mijn focus lag voornamelijk op het automatiseren van het deployen van een applicatie zodra er een nieuwe virtuele machine (VM) wordt aangemaakt, wat het hele proces efficiënter maakt.

Docker Project

Voor het vak Linux Webservices ontwikkelde ik in een Docker-project een schaalbare webpagina met een achterliggende databank. Het doel was om een webtoepassing te ontwerpen en implementeren die dynamische inhoud kan weergeven.

Aangezien Docker een volledig nieuw concept voor mij was, vergde het project veel opzoekwerk en ‘trial and error’. Toch slaagde ik erin het project succesvol af te ronden.

Tijdens dit proces heb ik in korte tijd veel geleerd over het opzetten en beheren van containers met Docker en Docker Compose, een ervaring die mijn technische kennis aanzienlijk heeft versterkt.

Azure Project

In deze opdracht was het de bedoeling om event logs en prestatiegegevens van een Azure Windows VM door te sturen naar Azure Monitor, met als doel een efficiënte monitoring en analyse op te zetten.

Wat ik zo waardeer aan de opdrachten binnen het vak Cloud Platforms, is dat je sterk wordt aangemoedigd om zelfstandig oplossingen te zoeken. Voor deze opdracht had ik nauwelijks ervaring met Azure of cloudtechnologie in het algemeen.

Intussen heb ik een goed inzicht verworven in de basisprincipes van Azure, en specifiek in het effectief inzetten van Azure Monitor. Ik zie nu ook duidelijk het belang van goede monitoring binnen een cloudomgeving.

Stage

Van Februari tot Mei 2025 heb ik stage gelopen bij Cegeka, hieronder kan u een overzicht vinden.

Introductie

Cegeka is een toonaangevend Europees IT-bedrijf dat organisaties helpt bij hun digitale transformatie. Met hoofdkantoor in Hasselt (België) biedt Cegeka oplossingen op het gebied van onder andere: cloudservices, cybersecurity, en softwareontwikkeling.

Tijdens mijn stage bij Cegeka werkte ik binnen het Detection Engineering Team van het Security Operations Center (SOC). Ik vergeleek en harmoniseerde detectieregels tussen Splunk en Microsoft Sentinel om detection gaps te verkleinen. Hiervoor ontwikkelde ik een methode waarmee 25 regels succesvol werden afgestemd en deels uitgerold bij klanten. Daarnaast ondersteunde ik bij threat hunts, detectieregelvalidaties en detectieregelmigraties tussen beide platformen.

Wilt u hier meer over weten? Neem dan gerust een kijkje in mijn plan van aanpak en realisatiedocument.

Realisatie

Mijn stage bestond uit drie fasen. Eerst inventariseerde ik alle detectieregels en onderzocht ik overlap tussen twee SIEM-omgevingen. De resultaten legde ik vast in een Excel-bestand genaamd 'Use-case library'. Aangezien dit vertrouwelijke informatie bevat, heb ik een mock-up versie gemaakt. Deze kan u bekijken via de knop 'Use-Case Library'.

In de tweede en derde fase heb ik de regels onderling vergeleken en op elkaar afgestemd, zodat beide omgevingen dezelfde detectielogica hanteren. Dit proces werd uitgevoerd aan de hand van een aligneerprocedure. Ook dit bestand bevat mock-up data. Klik op de knop 'Procedure' om deze te bekijken.

Tot slot stelde ik een reflectiedocument op over mijn stageperiode. Hierin beschrijf ik het verloop van de stage, de uitdagingen die ik ben tegengekomen en vooral, wat ik allemaal heb bijgeleerd bij Cegeka. U kan dit document bekijken via de knop 'Reflectie'
