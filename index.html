Wie ben ik?

Ik ben een techliefhebber met een passie voor het beveiligen van digitale omgevingen. Buiten het coderen geniet ik van gitaar spelen en het volgen van een strak fitnessschema.

Na het behalen van mijn bachelordiploma in Cloud & Cybersecurity droom ik ervan om aan de slag te gaan in een Security Operations Center (SOC). Daar wil ik mijn technische vaardigheden verder aanscherpen, ervaring opdoen in het analyseren van bedreigingen en me verdiepen in threat detection en incident response.

Mijn ultieme doel is om actief bij te dragen aan een veiliger digitaal landschap, waarin ik zowel mijn passie voor cybersecurity als mijn drive om impact te maken kan inzetten.

Stage Detection Engineer

Bij Cegeka liep ik drie maanden stage binnen het Detection Engineering-team. Tijdens deze periode werkte ik met SIEM-oplossingen zoals Splunk en Microsoft Sentinel, waarbij ik voornamelijk aan de slag ging met de querytalen SPL en KQL. Daarnaast heb ik mijn kennis verdiept in threat hunting en kreeg ik inzicht in de dagelijkse werking van een Cybersecurity Operations Center (SOC).

SOC Project

Voor mijn SOC-project, uitgevoerd in het kader van het vak Cybersecurity en Security Operations Center, heb ik zelf een volledig functionerend SOC opgebouwd. Dit SOC bestaat uit verschillende componenten, waaronder EDR met Suricata, een SIEM-oplossing met Wazuh, threat hunting en digital forensics met Velociraptor en penetratietesten met behulp van ART en Kali Linux.

Automatische Muizenval

Voor het vak IoT Essentials ontwikkelde ik een geavanceerde, slimme muizenval met behulp van Orange Pi.

De val reageert automatisch wanneer de ingebouwde sensor beweging detecteert binnen zijn bereik. Ze kan zichzelf resetten, en op het LCD-scherm wordt weergegeven hoe vaak de val is geactiveerd.

Dit was zonder twijfel een van mijn leukste projecten. Het was enorm boeiend om verschillende componenten via Arduino te laten samenwerken tot één goed functionerend geheel. Bovendien heb ik dankzij dit project veel bijgeleerd over elektriciteit en elektronica, een onderwerp dat me sindsdien nog meer fascineert.

Project Hosting

In mijn groepsproject voor het vak Project Hosting hebben we ons eigen hostingplatform gebouwd voor het draaien van applicaties, gebaseerd op onze eigen serverconfiguratie. Het concept is vergelijkbaar met het Okteto Cloud platform.

Dit was voor mij een compleet nieuw domein, wat het des te uitdagender en boeiender maakte.

Mijn focus lag voornamelijk op het automatiseren van het deployen van een applicatie zodra er een nieuwe virtuele machine (VM) wordt aangemaakt, wat het hele proces efficiënter maakt.

Docker Project

Voor het vak Linux Webservices ontwikkelde ik in een Docker-project een schaalbare webpagina met een achterliggende databank. Het doel was om een webtoepassing te ontwerpen en implementeren die dynamische inhoud kan weergeven.

Aangezien Docker een volledig nieuw concept voor mij was, vergde het project veel opzoekwerk en ‘trial and error’. Toch slaagde ik erin het project succesvol af te ronden.

Tijdens dit proces heb ik in korte tijd veel geleerd over het opzetten en beheren van containers met Docker en Docker Compose, een ervaring die mijn technische kennis aanzienlijk heeft versterkt.

Azure Project

In deze opdracht was het de bedoeling om event logs en prestatiegegevens van een Azure Windows VM door te sturen naar Azure Monitor, met als doel een efficiënte monitoring en analyse op te zetten.
